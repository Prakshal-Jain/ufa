- `e.preventDefault()`; set status "submitting".
- `const data = new FormData(e.currentTarget);`

- if `res.ok` -> status "success" (show a success message, optionally reset form); else -> status "error".
- wrap in try/catch -> status "error" on network failure.

States UI:
- success: replace form with a confirmation message ("Thanks - we'll be in touch." + the contact email as a mailto fallback link).
- error: keep the form, show an inline error with a `mailto:${site.contactEmail}` fallback link.
- submitting: disable the submit button + label it "Sending...".
